# [CRUD php]
## Descrição

[O projeto tem como o objetivo aprender a integrar php com HTML e css para 
iniciar as atividades na empresa IdeaTech. Foi feito um CRUD básico para
registrar alunos por nome, email e curso que participa atualmente]

## Funcionalidades

- Cadastrar aluno
- Editar dados do aluno aluno
- Deletar aluno

## Tecnologias

- php v 8.4
- CSS
- HTML 5


## Como Executar

1. Clone o repositório

2. Crie o banco de dados com base no script

3. Instale o apache2 pelo ubuntu, faça a conexão
com o repositório por meio de link e coloque o link 
em uma pasta do apache2 e coloque o comando
'sudo service apache2 start'

4. Abra o navegador e digite 'localhost' e selecione
a pasta com o link para os arquivos
